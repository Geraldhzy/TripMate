7. LOGGER API REFERENCE
═══════════════════════════════════════════════════════════════════════════════

Core Methods:
  logger.debug(message, data)  - DEBUG level log
  logger.info(message, data)   - INFO level log
  logger.warn(message, data)   - WARN level log
  logger.error(message, data)  - ERROR level log

Context Methods:
  logger.child({ key: value })  - Create scoped child logger

Timing Methods:
  const timer = logger.startTimer('operation')
  timer.done({ result: value })  - End timer and log
  timer.elapsed()                - Get elapsed time without logging

Utilities:
  logger.generateId()     - Generate 8-char random ID
  logger.formatDuration() - Format ms to human-readable (ms/s/min)
  logger.truncate(str)    - Safely truncate strings to 200 chars
