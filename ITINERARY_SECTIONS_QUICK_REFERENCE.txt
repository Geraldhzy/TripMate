================================================================================
                              HELPER FUNCTIONS
================================================================================

extractSegmentsByType(types)
  Input: Array<string> of segment types
  Output: Array of { day, date, city, segments[] } groups
  Used by: renderSectionTransport, renderSectionHotel, renderSectionFoodAndAttraction

renderExtractedSegments(groups, emptyIcon, emptyText)
  Renders grouped segments as cards
  Each card shows: Day N · Title + Location + Notes

renderDaysPlan(daysPlan)
  Renders collapsible day cards with expand/collapse buttons
  Each day can show timeline of segments

renderTimeline(segments)
  Renders vertical timeline with dots, lines, times, and content
  Used for daily activity sequences

renderBudgetSummary(summary)
  Renders budget breakdown table
  Shows each category, total, and remaining/overage

translateWeather(desc)
  Maps English weather descriptions to Chinese
  Examples: "Clear" → "晴", "Rainy" → "下雨"

translateCity(name)
  Maps English city names to Chinese
  Examples: "Tokyo" → "东京", "Bangkok" → "曼谷"

toggleReminder(el)
  Interactive handler for reminder checkboxes
  Toggles .checked class and shows/hides ✓ checkmark
