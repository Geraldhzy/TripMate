  Function: sendSSE(event, data)  [server.js line 120]
  
  Implementation:
    res.write(`event: ${event}\ndata: ${JSON.stringify(data)}\n\n`)
  
  Event Sequence (typical response):
  
    1. thinking              {} 
                            ↓ LLM is processing
    
    2. token x100           {text: "我会帮你规划..."}
                            ↓ Streamed chunks (real-time typing)
    
    3. thinking_done        {}
                            ↓ LLM finished streaming
    
    4. tool_start           {id: "call_1", name: "update_trip_info"}
                            ↓ Tool executing
    
    5. tool_result          {id: "call_1", name: "update_trip_info", 
                             resultLabel: "已记录目的地、出行时间"}
                            ↓ Tool done
    
    6. tripbook_update      {constraints: {...}, itinerary: {...}}
                            ↓ Panel update
    
    7. round_start          {round: 2}
                            ↓ Next iteration indicator
    
    8. agents_batch_start   {count: 2, agents: [{agent: "flight", 
                             label: "✈️ Flight Search", ...}]}
                            ↓ Sub-agent delegation
    
    9. agent_start          {agent: "flight", label: "✈️ Flight Search"}
                            ↓ Individual agent starting
    
    10. agent_success       {agent: "flight", data: {...}}
                            ↓ Agent completed
    
    11. token x50           {text: "根据搜索结果..."}
                            ↓ More LLM text
    
    12. quick_replies       {questions: ["选项1", "选项2"]}
                            ↓ Suggested next steps
    
    13. done                {}
                            ↓ Response complete
    
    ❌ error               {message: "..."}
                            ↓ Error occurred (early termination)
